Leaflet Project - GIS 
A modern rental map showcase built with Leaflet and OpenStreetMap. This project presents a responsive city listings grid alongside an interactive map for prominent Indian IT hub cities, with marker tooltips and hover-driven map zoom for a dynamic property browsing experience.

Live Demo
🔗 Demo: https://leaflet-smoky.vercel.app/

Features
- Interactive Leaflet map with OpenStreetMap tiles
- City rental cards for Hyderabad, Bangalore, Chennai, Gurugram, New Delhi, and West Bengal
- Marker tooltips showing rent, area, and room count
- Hover-to-zoom behavior linking city cards to map locations
- Responsive layout with grid-based city cards and mobile adaptive styling
- Modular JavaScript for map initialization, markers, popups, and event handling

Project Goal
The goal of this project is to build an interactive, visually engaging rental market overview using Leaflet and plain JavaScript. It is designed to demonstrate map-based data visualization, responsive layout, and basic UI-driven interactivity for a property listings dashboard.

Installation
Clone the repository:

```bash
git clone https://github.com/<your-username>/leaflet-rental-map.git
```

Open `index.html` in a browser or serve the folder with a static server.
